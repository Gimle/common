Welcome to Gimlé Common Extension
=================================

Under development.


Requirements
------------
php 5.3 / 5.4


Installation
------------
Download or checkout from reposirtory to a non-public folder.

Add this to the core ini file to load globaly, or to a project ini file to load for that project only.
common = "/absoulute/path/to/gimle/common/extension/"
